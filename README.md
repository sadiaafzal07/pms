Pharmacy Management System – Frontend

This repository contains the frontend of a Pharmacy Management System (PMS) developed using Next.js. It provides an interface for managing key pharmacy operations such as customers, inventory, orders, and refills.


Features

•	Dashboard with overview and analytics

•	Customer management (add, edit, view, delete)

•	Inventory management

•	Order management

•	Refill tracking

•	Settings page

•	Search, filtering, and pagination


Tech Stack

•	Next.js (App Router)

•	React

•	TypeScript

•	Tailwind CSS

•	Lucide Icons


Project Structure
app/                → Routing and pages (Next.js App Router)

components/         → Reusable components

components/ui/      → UI elements

components/data/    → Mock data (temporary)

public/             → Static assets


Installation and Setup

1.	Clone the repository:
   
git clone https://github.com/HafsaAmir19/pharmacy-management-system.git

2.	Navigate to the project folder:
   
cd pharmacy-management-system

3.	Install dependencies:
   
npm install

4.	Run the development server:
   
npm run dev

5.	Open the application in your browser:
   
http://localhost:3000


Notes

•	Data is handled using mock data.
